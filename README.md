💎 HiddenGem

HiddenGem is a web platform that helps users discover underrated local businesses, cafes, shops, and services that are often overlooked on major platforms.
Instead of showing only popular places, HiddenGem focuses on real local gems shared by the community.

🌐 Live App:
https://hidden-gem-frontend.onrender.com

The application is hosted on Render and uses a modern React-based frontend architecture.

🚀 Features
🔎 Discover Hidden Places

Find underrated local businesses near you

Explore cafes, kirana stores, xerox shops, restaurants, and more

🏪 Add Local Gems

Users can submit places that deserve recognition

Community-driven discovery

📍 Location-Based Listings

Businesses are displayed with location information

Helps users find gems in their area

⭐ Community Focus

Encourages people to support small local businesses

Promotes neighborhood entrepreneurship

⚙️ Installation (Local Setup)
1️⃣ Clone Repository
git clone https://github.com/maniiscursed/HiddenGem.git
cd HiddenGem
2️⃣ Backend Setup

Install dependencies:

pip install -r backend/requirements.txt

Run backend server:

uvicorn backend.main:app --reload

Backend will run at:

http://127.0.0.1:8000
3️⃣ Frontend Setup

Navigate to frontend:

cd frontend
npm install
npm run dev

Frontend runs at:

http://localhost:5173
🌍 Deployment

The project is deployed using Render:

Frontend → Render Static Service

Backend → Render Web Service

Database → Render PostgreSQL
